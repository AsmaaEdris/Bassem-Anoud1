# Bassem & Anoud — Wedding Invitation

A luxury digital wedding invitation website for Bassem & Anoud's marriage celebration on 13 June 2026.

## Features

- **Envelope animation** — the site opens as a sealed envelope; clicking/tapping it unfolds with a smooth animation to reveal the invitation
- **Royal & gold theme** — deep burgundy `#760031` paired with warm gold, inspired by classic luxury wedding stationery
- **Great Vibes + Montserrat** typography — a flowing script font for headers paired with clean sans-serif for body text
- **Live countdown** — real-time countdown to 13 June 2026, 7:00 PM Cairo time (UTC+3)
- **Couple photo** — displayed in a decorative circular gold frame with animated ring
- **Quranic verse** — displayed in Arabic with right-to-left layout
- **Venue button** — links directly to Google Maps for Opera Hall, Aero Sport Club
- **Auto-playing background music** — loops throughout the visit
- **Floating gold particles** — ambient animated particles for atmosphere

## Tech Stack

- [TanStack Start](https://tanstack.com/start) (React SSR framework)
- [Tailwind CSS v4](https://tailwindcss.com)
- [Vite](https://vitejs.dev)
- Deployed on [Netlify](https://netlify.com)

## Running Locally

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:3000`.

For Netlify feature emulation (edge functions, etc.):

```bash
netlify dev
```
